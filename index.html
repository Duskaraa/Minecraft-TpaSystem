🌍 TPASystem

A teleport request system for Minecraft Bedrock Edition

Features Installation Commands Configuration License

✨ Features

  • 🔁 /tpa <player> → Send a teleport request
  • ✅ /tpaccept → Accept last request
  • ❌ /tpdeny → Deny last request
  • ⏳ Auto-expiring requests (30s default)
  • 🔒 Optional permission system
  • 💬 Customizable messages & sounds

📥 Installation

  1. Download or clone this repository:
    2. git clone https://github.com/usuario/TPASystem.git
  2. Copy behavior_packs/ and resource_packs/ into your Minecraft world folder.
  3. Enable both packs in world settings.
  4. Turn on Experimental Scripting Features.

💻 Commands

Command Description
/tpa <player> Send a teleport request
/tpaccept Accept last request
/tpdeny Deny last request

⚙️ Configuration

Edit main.js to customize:

// Request timeout
const REQUEST_TIMEOUT = 30000;

// Messages
const messages = {
  requestSent: "📨 You sent a request to %player%",
  requestReceived: "📥 %player% wants to teleport to you",
  accepted: "✅ You accepted %player%'s request",
  denied: "❌ You denied %player%'s request",
  expired: "⌛ The teleport request expired"
};

// Example permission system
function hasPermission(player, permission) {
  return player.hasTag(permission);
}
